**РентКар** – это полнофункциональная платформа для краткосрочной и долгосрочной аренды автомобилей.  
Проект объединяет **арендаторов**, желающих быстро найти и забронировать авто, **владельцев** автомобилей, которые сдают свой транспорт в аренду, и **администраторов**, обеспечивающих работу системы.

Сайт разработан на фреймворке **Yii2 (Basic Template)** с использованием **Tailwind CSS**, **MySQL** и современных JavaScript-решений (AJAX, календарь, галерея, графики).

## ✨ Возможности

### Для всех пользователей (гости)
- **Главная страница** с поиском по датам, популярными авто, преимуществами и промо-блоками.
- **Каталог автомобилей** с фильтрацией (марка, цена, год, тип кузова, КПП, объём двигателя, привод, топливо, расход, опции) и сортировкой (по цене, году, рейтингу).
- **Детальная страница автомобиля**: фотогалерея, полные характеристики, календарь занятости, отзывы, кнопка бронирования.
- **Форма входа/регистрации**, восстановление пароля.
- **Страницы** «О нас», «Контакты», «Правила аренды».

### Для арендаторов (роль `user`)
- **Личный кабинет** с редактированием профиля (аватар, ФИО, телефон, email).
- **Мои бронирования** – список текущих и завершённых заказов, возможность отмены (для статусов `pending`/`confirmed`).
- **Отзывы** – после завершения аренды можно оставить оценку и комментарий.
- **Избранное** – добавление/удаление понравившихся автомобилей (AJAX).
- **Просмотр деталей бронирования** с полной информацией.

### Для владельцев автомобилей (роль `owner`)
- **Управление своими автомобилями**: создание, редактирование, удаление.
- **Загрузка нескольких фотографий** (первая становится главной).
- **Управление характеристиками** (список особенностей: кондиционер, подогрев, навигатор и т.д.).
- **Просмотр заявок на бронирование** с возможностью подтверждения или отклонения.
- **Календарь занятости** для каждого авто.
- **Статистика** по автомобилю: доход, количество бронирований, график по месяцам.

### Для администратора (роль `admin`)
- **Дашборд** с общей статистикой (пользователи, авто, бронирования, доход).
- **Управление пользователями** – смена ролей, редактирование, удаление.
- **Управление автомобилями** – модерация, редактирование, удаление любых объявлений.
- **Управление брендами и моделями** (CRUD).
- **Управление характеристиками** (CRUD) – например, «ABS», «Круиз-контроль», «Детское кресло».
- **Управление бронированиями** – просмотр, смена статуса, удаление.
- **Управление отзывами** – удаление нежелательных.
- **Расширенная статистика** – доход по месяцам, популярные автомобили, активность по дням недели, лучшие клиенты.

### Сквозной функционал
- **Бронирование автомобиля**:
  - Выбор дат через календарь (с проверкой доступности).
  - Расчёт стоимости (цена за день × количество дней + расширенная страховка, если выбрана).
  - Ввод данных водителя, паспортных данных, страховки.
  - Создание заказа со статусом `pending` (ожидает подтверждения владельца).
- **Динамический календарь** (Date Range Picker) на главной и на странице авто.
- **Интерактивный календарь занятости** (подсветка занятых дат).

## ⚙️ Установка

### Требования
- PHP 7.4 или выше
- MySQL 5.7 / MariaDB 10.4+
- Composer
- Node.js (для сборки Tailwind, опционально)

### Шаги установки

1. **Клонировать репозиторий**
   ```bash
   git clone https://github.com/yourusername/rent-car.git
   cd rent-car
   ```

2. **Установить зависимости Composer**
   ```bash
   composer install
   ```
3. **Настроить базу данных**

- Создайте пустую БД (например, rent_car).
- Импортируйте дамп: *mysql -u root -p rent_car < rent-car.sql* (файл rent-car.sql из корня).
- Либо выполните миграции, если они подготовлены (в данном проекте используется SQL-дамп).

4. **Настроить подключение к БД**
Отредактируйте *config/db.php*:

```php
return [
    'class' => 'yii\db\Connection',
    'dsn' => 'mysql:host=localhost;dbname=rent_car',
    'username' => 'root',
    'password' => '',
    'charset' => 'utf8',
];
```
5. **Установить cookieValidationKey**
В *config/web.php* вставьте случайную строку:

```php
'request' => [
    'cookieValidationKey' => 'ваш_уникальный_ключ',
],
```

6. **Запустить веб-сервер**

```bash
php yii serve
```

## 👥 Роли и права доступа

| Роль           | Права                                                                      |
|----------------|----------------------------------------------------------------------------|
| guest          | Просмотр главной, каталога, карточек авто, страниц входа/регистрации.      |
| user           | Все права гостя + личный кабинет, бронирование, избранное, отзывы.         |
| owner          | Все права user + управление своими автомобилями, заявками, статистика.     |
| admin          | Полный доступ ко всем функциям, включая админ-панель и управление системой.|


Роль определяется полем *role* в таблице *users*. Контроллеры используют *AccessControl* с *matchCallback*.

## 📦 Основные модули

**🚘 Каталог и поиск**

- Контроллер: *CatalogController*
- Маршрут: */catalog/index*
- Фильтрация через GET-параметры, пагинация, сортировка.

**🏎️ Детальная страница автомобиля**

- Контроллер: *CarController*
- Маршрут: */car/view?id=xxx*
- Календарь занятости (JavaScript + проверка через модель Booking).
- Избранное (только для авторизованных).

**📅 Бронирование**

- Контроллер: *BookingController*
- Маршрут: */booking/create (POST)*, */booking/confirm (GET/POST)*.
- Этапы: выбор дат → форма ввода данных → сохранение → просмотр.
- Расчёт цены с учётом страховки (basic/extended).

**👤 Личный кабинет арендатора**

- Контроллер: *CabinetController*
- Действия: *profile*, *bookings*, *favorites*, *cancelBooking*, *createReview*, *viewReview*.

**🧑‍💼 Личный кабинет владельца**

- Контроллер: *OwnerController*
- Действия: *index (дашборд)*, *cars*, *create*, *update*, *delete*, *bookings*, *confirmBooking*, *cancelBooking*, *calendar*, *stats*.

**🛠️ Административная панель**

- Контроллер: *AdminController*
- Разделы: пользователи, автомобили, бренды, модели, характеристики, изображения, бронирования, отзывы, статистика.

## 🗃️ Схема базы данных

Основные таблицы:

| Таблица                 | Описание                                                                             |
|-----------------------  |--------------------------------------------------------------------------------------|
| users                   | Пользователи: id, username, email, password, role, avatar и т.д.                     |
| car_brands              | Бренды автомобилей (Toyota, BMW и т.п.).                                             |
| car_models              | Модели с привязкой к бренду (brand_id).                                              |
| cars                    | Объявления об автомобилях: владелец, модель, характеристики, цены, статус.           |
| car_images              | Изображения авто (car_id, image_path, is_main).                                      |
| car_features            | Список возможных особенностей (например, «Кондиционер»).                             |
| car_feature_assignments | Связь автомобилей с особенностями (many-to-many).                                    |
| bookings                | Бронирования: автомобиль, пользователь, даты, итоговая цена, статус, данные водителя.|
| payments                | Платежи (интеграция с платёжным шлюзом).                                             |
| reviews                 | Отзывы к бронированиям (рейтинг, комментарий).                                       |
| favorites               | Избранное: связь user_id – car_id.                                                   |
| contacts                | Сообщения из формы обратной связи.                                                   |
| services                | Дополнительные услуги (детское кресло, навигатор, расширенная страховка).            |

Полная структура с индексами и внешними ключами приведена в файле *rent-car.sql*.
